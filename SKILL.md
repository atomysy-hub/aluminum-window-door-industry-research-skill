### Step 5 — 创建腾讯文档智能文档（最终输出）

**⚠️  重要：mcporter CLI 在 Windows PowerShell 下因中文编码问题无法传递 JSON，必须用 Node.js 脚本直接发 HTTP 请求。**

#### 5.1 获取 Token 和 Sign

从 mcporter 配置中读取（Token 会话期有效，无需每次重新授权）：

```bash
mcporter config get tencent-docs
```
提取 `Authorization:` 的值（32位token）和 `X-MCP-Request-Sign:` 的值（`q` + 64位SHA256）。

#### 5.2 用 Node.js 脚本创建文档

将以下脚本保存为 `call_tdoc_mcp.js`，运行 `node call_tdoc_mcp.js`：

```javascript
// call_tdoc_mcp.js
const https = require('https');
const crypto = require('crypto');
const fs = require('fs');

// 1. 从 mcporter config 读取 token 和 sign
const { execSync } = require('child_process');
const configOut = execSync('mcporter config get tencent-docs 2>&1', { encoding: 'utf8' });
const token = (configOut.match(/Authorization:\s*([^\n]+)/i) || ['',''])[1].trim();
const existingSign = (configOut.match(/X-MCP-Request-Sign:\s*([^\n]+)/i) || ['',''])[1].trim();
const sign = existingSign || ('q' + crypto.createHash('sha256').update(token + 'qclaw').digest('hex'));

// 2. 读取 JSON 参数文件（MDX 内容）
const args = JSON.parse(fs.readFileSync('temp_doc_args.json', 'utf8').trim().replace(/^\uFEFF/, ''));

// 3. 构造 JSON-RPC 2.0 请求
const body = JSON.stringify({
  jsonrpc: '2.0', id: Date.now(), method: 'tools/call',
  params: { name: 'create_smartcanvas_by_mdx', arguments: { title: args.title, content_format: 'mdx', mdx: args.mdx } }
});

// 4. 发送请求
const req = https.request({
  hostname: 'docs.qq.com', path: '/openapi/mcp', method: 'POST',
  headers: { 'Content-Type': 'application/json', 'Authorization': token, 'X-MCP-Request-Sign': sign, 'Content-Length': Buffer.byteLength(body) }
}, res => { let d=''; res.on('data', c=>d+=c); res.on('end', () => { const r=JSON.parse(d); console.log('file_id:', r.result?.structuredContent?.file_id); console.log('url:', r.result?.structuredContent?.url); }); });
req.on('error', e => console.error(e)); req.write(body); req.end();
```

#### 5.3 参数文件格式（temp_doc_args.json）

```json
{
  "title": "铝合金门窗行业动态_YYYYMMDD",
  "content_format": "mdx",
  "mdx": "【完整MDX内容】"
}
```

# 铝合金系统门窗行业动态调研

<Callout type="info">
本报告聚焦2025-2026年铝合金系统门窗行业关键动态，涵盖市场规模、政策新规、技术趋势、竞品动向及原材料行情五大维度。
</Callout>

## 一、行业概况

### 1.1 市场规模

2025年中国门窗行业市场规模已突破 **8800亿元**，其中主打系统节能、静音防护、高端定制的中高端产品市场占比已接近六成。

豪美新材2025年年报：全年营收 **77.78亿元**（同比+16.58%），铝型材销量 **30.74万吨**（同比+15.36%）。

### 1.2 市场格局

| 梯队 | 代表品牌 | 定位 |
|------|---------|------|
| 高端外资 | 旭格(Schüco)、贝克洛、YKK AP | 高端系统门窗，工程项目为主 |
| 国内一线 | 皇派、墨瑟(MOSER)、派雅、希洛、良木道 | 中高端系统门窗，零售+工程双轨 |
| 成长型 | 新豪轩、百利玛、轩尼斯、欧哲 | 规模化+差异化并重 |

### 1.3 铝合金产量数据

- 2025年全年中国铝合金累计产量 **1929.7万吨**，累计增长 **15.8%**
- 2026年1-2月铝合金产量 **276.5万吨**，累计增长 **8.9%**

## 二、政策动态

### 2.1 国标强制落地：门窗设计工作年限不低于25年

2025年5月1日《住宅项目规范》正式实施，**"建筑外窗设计工作年限不低于25年"** 首次以条文形式写入国家规范。门窗从"装修耗材"正式升级为"安全底线"。

4月15日，CECS《铝合金门窗安全构造技术要求》团体标准宣贯会在广东佛山举行，明确了门窗安全构造技术规范。

### 2.2 政策影响分析

- **门槛提升**：不达标的中小品牌将被加速淘汰，行业集中度提升
- **品质溢价**：符合25年标准的门窗产品将获得品牌溢价空间
- **检测认证**：第三方检测认证需求增加

## 三、技术趋势

### 3.1 材料创新

- **断桥铝合金 + Low-E玻璃**：主流节能方案，保温隔声性能大幅提升
- **铝包木门窗**：高端市场持续增长，2025年市场规模约 **95.5亿元**，智能化、低碳化、全球化三大主线明确

### 3.2 智能化趋势

智能门窗可通过手机APP远程控制、感应开启等产品逐步成熟，个性化与智能化成为未来发展方向。

### 3.3 绿色建筑驱动

随着绿色建筑推广，门窗更注重环保材料使用和能源效率提升。

## 四、竞品动态

### 4.1 皇派门窗（近期重点）

- 荣膺"**信用5A、质量5星、服务5星**"企业殊荣
- 中国系统门窗标准制定单位之一，深耕行业近十九载
- 两大智能生产基地，面积超 **22万平方米**，CNAS检测实验中心
- 工信部授予"**国家绿色工厂**"称号
- 专卖店 **800+家**，覆盖30个省份
- 正在推进深交所上市

### 4.2 墨瑟MOSER

- 源于德国，1995年成立，2005年进入中国
- 主打实木窗、铝包木门窗、断桥铝门窗
- 占地 **5.5平方公里** 国家建筑节能技术创新园
- 中国新型节能门窗行业标准主编单位

### 4.3 行业活动

广东（南海）铝加工产业联盟组织20+会员企业参加第九届建筑系统门窗年会，破解建筑铝型材行业发展困境。

## 五、原材料行情

### 5.1 铝锭价格走势

| 时间 | 长江有色均价（元/吨） | 趋势 |
|------|-------------------|------|
| 2025年底 | ~23193 | 全年上涨 |
| 2026年1月 | 持续上行 | 突破阶段高位 |
| 2026年4月 | 24500-24800 | 高位震荡 |

近期（4月）沪铝主力合约在 **24500-25500元/吨** 区间震荡。广发期货观点：海外低库存高升水与国内高库存影响下，上下行空间均有限，短期高位盘整，中长期趋势偏多。

### 5.2 影响分析

- 铝锭高位运行压缩中小门窗企业利润空间
- 推动行业向高附加值产品（系统门窗、定制化）转型
- 一体化生产、品牌溢价能力成为抗压关键

## 六、趋势研判（未来6个月）

1. **25年国标效应加速发酵**：不达标品牌加速退出，一线品牌市占率进一步提升
2. **铝价高位震荡偏强**：原材料成本压力持续，中小品牌整合提速
3. **旧房换新成零售增长主引擎**：存量房市场激活，改善型需求释放
4. **综合实力竞争时代**：技术研发、智造产能、品牌服务、全场景解决方案缺一不可
5. **出口贸易摩擦影响**：美国对铝制品双反调查持续，关注内需市场布局
